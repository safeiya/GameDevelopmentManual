# 游戏开发之路食用指南 2019.10.23ver
## 开始
- 程序：选择一个游戏引擎、学习并入门引擎支持的编程语言。要做什么类型的游戏，就改类型学习工程示例Demo。
- 美术：选择一个绘画、图片处理工具，有条件一定要买一个数位板。锻炼绘画功底，学习一些项目规范。
- 策划：学会程序或者美术二选一入门之后转策划，这样才能在项目中有话语权。美术偏文案方面，程序偏玩法放面。
